# sec2time
標準入力から秒数を読み込み、`分:秒` の形式に変換して出力するコマンドです。
## インストール方法
以下のコマンドを入力してください。

'''
$git clone https://github.com/shogo05/robosys2025.git 
'''

'''
$cd robosys2025
'''
